## Numbers

## Hexadecimal numbers

```js
0xf3; // hexadecimal for: 243
```

## Expotential numbers

```js
var a = 5e10;
a; // 50000000000
a.toExponential(); // "5e+10"

var onethousand = 1e3; // means 1 * 10^3
var onemilliononehundredthousand = 1.1e6; // means 1.1 * 10^6
```

## Octal numbers

```js
0o363; // octal for: 243
```

## Binary numbers

```js
0b11110011; // binary for: 243
```
